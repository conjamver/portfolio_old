Mellow Me Out - Capstone Project

Introduction

This project required building a fully functional website for the Mellow Me Out brand. My role in the project was UX/UI developer. The main goal was to showcase what the buisiness offers and providing a flexible booking system for the client. The design of the website had to be up to date with its competitors.

Assertion

My role as the UX/UI developer required me creating high-fidelity wireframes in photoshop and then proceeding to code the front-end design using frameworks such as Bootstrap 4 and jQuery. By incorporating usability principles, this ensured that the website would be appealing to the users that want to know more about the business.

My contribution to the meetings were discussing ideas that may help the website, following up the progress of website and what tasks that needed to be completed.
